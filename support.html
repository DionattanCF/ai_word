Perguntas Frequentes

Como configurar o JusTexto?

Para configurar o JusTexto, você precisa inserir sua chave API da OpenAI e adicionar pelo menos um assistente. Consulte nossa página inicial para obter instruções detalhadas.

Como criar meu assistente personalizado?

Você pode criar assistentes personalizados na plataforma OpenAI. Acesse https://platform.openai.com/assistants e siga as instruções para criar um assistente com instruções específicas para suas necessidades.

O JusTexto funciona offline?

Não, o JusTexto requer uma conexão com a internet para se comunicar com a API da OpenAI e processar o texto.
