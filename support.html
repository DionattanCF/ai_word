Perguntas Frequentes

Como posso começar a usar o JusTexto?

Após instalar o add-in JusTexto, abra o Microsoft Word e clique no botão JusTexto na guia Início. Configure sua chave API da OpenAI e comece a usar o assistente para otimizar seus textos jurídicos.

Quais funcionalidades o JusTexto oferece?

O JusTexto permite reescrever, resumir, simplificar e criar contra-argumentos para seus textos jurídicos. Também é possível fazer perguntas específicas para o assistente através do chat integrado.

Posso configurar o JusTexto para necessidades específicas?

Sim, o JusTexto permite adicionar diferentes assistentes pré-configurados na OpenAI, possibilitando a customização para diferentes tipos de tarefas jurídicas.

Política de Privacidade

O JusTexto não armazena nenhum texto ou documento enviado pelo usuário. Todas as interações com a API da OpenAI são realizadas utilizando sua própria chave API, garantindo que você mantenha controle total sobre seus dados.
