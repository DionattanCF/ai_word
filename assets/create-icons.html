Instruções para Criar Ícones para o JusTexto

Para finalizar a configuração do seu add-in JusTexto, você precisará criar e salvar ícones em vários tamanhos. Aqui estão as instruções passo a passo:

1. Crie os Ícones

Você pode criar ícones usando diversas ferramentas online gratuitas:

  • Favicon.io - Permite criar ícones a partir de texto, imagens ou emojis
  • Transparent Icon Maker - Para ícones com fundo transparente
  • PNG2ICO - Converte imagens PNG para o formato ICO

2. Tamanhos Necessários

Certifique-se de criar e salvar os ícones nos seguintes tamanhos:

  • icon-16.png - 16x16 pixels
  • icon-32.png - 32x32 pixels
  • icon-80.png - 80x80 pixels
  • icon-128.png - 128x128 pixels

3. Salvar os Ícones

Salve todos os ícones na pasta assets/ do seu projeto.

4. Hospedar os Ícones

Após criar os ícones, você precisará:

  1. Fazer upload de todos os arquivos do seu projeto para o domínio https://aiword.dionattan.adv.br/
  2. Certificar-se de que os arquivos de ícone estão disponíveis nos seguintes URLs:
    • https://aiword.dionattan.adv.br/assets/icon-16.png
    • https://aiword.dionattan.adv.br/assets/icon-32.png
    • https://aiword.dionattan.adv.br/assets/icon-80.png
    • https://aiword.dionattan.adv.br/assets/icon-128.png
Nota importante: Até que você complete o deploy do seu site, você pode usar URLs temporárias para os ícones no arquivo manifest.xml, como foi feito. Após o deploy completo, você pode atualizar o manifest.xml para apontar para os ícones hospedados em seu próprio domínio.

5. Atualizar o Manifest

Uma vez que os ícones estejam hospedados e acessíveis, atualize o arquivo manifest.xml para apontar para os URLs corretos dos ícones.
